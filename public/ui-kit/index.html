Карточки stable

Базовый блок каталога. Минимальная высота 170px, flex-column с gap 14px. Ссылки прижаты к низу через margin-top: auto на .card__links.

.card — default

Полный вид: логотип + мета (заголовок + описание) + ссылки. При ошибке загрузки логотипа показывается .card__logo-fallback с инициалами и акцентным фоном.

.card.card--skeleton — loading state

Показывается пока systems.json не загружен. Анимация card-sk-pulse: opacity 1→0.4→1 за 1.4s. Каждый элемент имеет свой animation-delay для волнового эффекта. pointer-events: none, user-select: none, aria-hidden="true".

Пустое состояние stable

Отображается когда в выбранной категории (Отечественные / Зарубежные) нет ни одной карточки. Управляется атрибутом hidden через JS.

В этой категории пока нет карточек.

.empty

Border: 1px dashed rgba(255,255,255,.22), border-radius --radius, padding 14px 16px. JS переключает атрибут hidden на элементе в зависимости от количества видимых карточек.
